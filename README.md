# 🔨 Starter repository for modern C++ libraries

This repository includes a starter project for modern C++ libraries, which uses the following tools and patterns:
- [CMake](https://cmake.org) for cross-platform builds
- [Conan2](https://docs.conan.io/2/index.html) for dependency management
- [Doxygen](https://www.doxygen.nl) for documentation
- [GoogleTest](https://github.com/google/googletest) for unit testing
- [Codecov](https://about.codecov.io) for code coverage
- [GitHub Actions](https://docs.github.com/en/actions) for continuous integration
- [Pitchfork layout](https://api.csswg.org/bikeshed/?force=1&url=https://raw.githubusercontent.com/vector-of-bool/pitchfork/develop/data/spec.bs) for directory structure
